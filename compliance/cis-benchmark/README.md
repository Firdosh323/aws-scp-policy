# CIS Benchmark Mapping

Representative policy coverage for AWS foundation benchmark goals:

- `deny-root-user.json`: discourages use of the root principal.
- `deny-disable-cloudtrail.json`: protects centralized logging.
- `deny-public-s3.json`: aligns with public access prevention practices.
- `deny-security-group-open.json`: aligns with network exposure hardening.
- `mandatory-tagging.json`: supports governance and inventory practices.

Review benchmark version requirements against current AWS guidance before using this mapping as evidence.
