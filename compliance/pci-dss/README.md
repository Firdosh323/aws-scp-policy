# PCI DSS Mapping

Representative policy coverage for payment environments:

- `deny-disable-cloudtrail.json`: supports audit logging preservation.
- `deny-public-s3.json`: helps reduce public data exposure risk.
- `deny-unencrypted-ebs.json`: supports storage encryption requirements.
- `enforce-encryption.json`: supports encryption-at-rest controls.
- `mandatory-tagging.json`: helps asset ownership and environment classification.

Use this mapping as implementation guidance, not as a formal certification statement.
