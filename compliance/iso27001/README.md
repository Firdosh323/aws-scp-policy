# ISO 27001 Mapping

Representative policy coverage for ISO 27001-aligned governance:

- `deny-disable-cloudtrail.json`: supports logging and monitoring governance.
- `deny-public-s3.json`: supports access-control and data-protection objectives.
- `deny-unencrypted-ebs.json`: supports cryptographic control objectives.
- `enforce-encryption.json`: supports technical encryption enforcement.
- `deny-leaving-org.json`: helps maintain centralized governance boundaries.

These mappings are illustrative and should be tied to your statement of applicability and control library.
