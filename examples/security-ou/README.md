# Security OU Baseline

Recommended attachment set for logging, audit, and security tooling accounts:

- `deny-root-user.json`
- `deny-leaving-org.json`
- `deny-disable-cloudtrail.json`
- `deny-public-s3.json`
- `deny-unencrypted-ebs.json`
- `deny-wildcard-iam-policies.json`
- `restrict-admin-role-creation.json`
- `enforce-encryption.json`

Avoid applying broad cost or region restrictions here until you confirm they do not interfere with security tooling.
