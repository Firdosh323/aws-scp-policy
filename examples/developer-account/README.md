# Developer Account Baseline

Recommended attachment set for individual or shared developer accounts:

- `deny-root-user.json`
- `deny-leaving-org.json`
- `deny-disable-cloudtrail.json`
- `deny-public-s3.json`
- `deny-unencrypted-ebs.json`
- `deny-security-group-open.json`
- `deny-wildcard-iam-policies.json`
- `mandatory-tagging.json`
- `restrict-expensive-resources.json`
- `restrict-expensive-regions.json`

Use this profile when you want broad freedom with a firm floor on security and cost risk.
